Danh mục lưu trữ
Repository này bao gồm:

* **`exercises/`**: Các bài tập thực hành cá nhân và nhóm.
* **`src/`**: Mã nguồn các dự án/thuật toán AI.
* **`reports/`**: Tài liệu báo cáo môn học.
* **`data/`**: Các tập dữ liệu phục vụ huấn luyện và kiểm thử.
* **`submissions/`**: Các file nộp cho giảng viên.
